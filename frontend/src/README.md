# proyecto_django1

## Descripción
Sistema de gestión Web CRUD (Crear, Leer, Actualizar, Eliminar) desarrollado como proyecto escolar. Esta aplicación conecta una interfaz de usuario en el Frontend desarrollada con React a un servidor Backend dinámico construido con Django.

---

## Tecnologías Utilizadas
* **Backend:** Python, Django, SQLite3
* **Frontend:** React, HTML5, CSS3, JavaScript
* **Control de Versiones:** Git & GitHub

---

## Instrucciones de Instalación

1. Clonar el repositorio:
   git clone https://github.com/cris-to516/proyecto_django1.git

2. Configurar el Backend (Django):
   python -m venv venv
   python manage.py migrate

3. Configurar el Frontend (React):
   cd Interfaz
   npm install

---

## Forma de Ejecución

1. Iniciar el servidor Backend (Django):
   python manage.py runserver

2. Iniciar la aplicación Frontend (React):
   cd Interfaz
   npm start
